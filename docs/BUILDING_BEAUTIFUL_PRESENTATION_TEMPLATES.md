What this implies

Gamma uses a block-based global theme engine, not slide-by-slide formatting like PowerPoint.

✅ 2. Galaxy Images Grid – AI Image Suggestion Panel

This is the AI-powered media suggestion panel.

What it does

Detects your topic (“Galaxy formation”).

Automatically proposes relevant images (galaxies, nebulae, clusters).

Pulls images from stock libraries (Unsplash, Pexels).

Lets users click any image to instantly apply it to a block.

Implied

Gamma has context-aware visual suggestions integrated directly into the editor.
